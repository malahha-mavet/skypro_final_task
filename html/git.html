Git

  • git status - просотр изменений в файле
  • git add имя файла- для добавления в отслеживаемые файлы.
  • git status - проверка статуса файла
  • git commit -m "текст" — сделаем коммит. Так мы покажем Git, что файл изменен. Изменения файла сохранятся в репозитории.
  • git push- внесение изменений на удалённый репозиторий
